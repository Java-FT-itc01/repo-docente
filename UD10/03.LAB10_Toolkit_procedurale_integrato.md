# 03. LAB10 - Toolkit procedurale integrato

## Obiettivo del laboratorio

In questo laboratorio costruirai una milestone procedurale integrata.

Il programma finale si chiamerà:

```text
ToolkitProceduraleIntegrato
```

Il programma avrà:

- un menu principale;
- un'area numerica;
- un'area stringhe;
- utility di input robusto;
- metodi statici per ogni responsabilità;
- gestione dello stato del programma;
- test documentati.

Questa milestone chiude il primo blocco procedurale del corso.

---

## 1. Prerequisiti

Prima di iniziare dovresti conoscere:

- `Scanner`;
- `nextLine()`;
- `Integer.parseInt(...)`;
- `try/catch`;
- `if`;
- `switch`;
- `while`;
- `do-while`;
- `for`;
- metodi statici;
- stringhe;
- array;
- ricerca;
- conteggio;
- minimo e massimo;
- media;
- ordinamento;
- `Arrays.toString(...)`;
- `Arrays.copyOf(...)`.

## 3. Menu principale

Il programma avrà questo menu principale:

```text
===== TOOLKIT PROCEDURALE INTEGRATO =====
1. Area numerica
2. Area stringhe
0. Esci
```

L'area numerica gestirà un array di interi.

L'area stringhe gestirà una stringa.

---

## 4. Stato del programma

Nel `main` useremo:

```java
int[] numeri = null;
String testo = null;
```

All'inizio non sono ancora disponibili dati.

Dopo l'inserimento:

- `numeri` conterrà l'array inserito;
- `testo` conterrà la stringa inserita.

I metodi devono controllare se i dati sono disponibili prima di usarli.

Segui questo ordine:

1. crea la classe e il `main`;
2. aggiungi i menu;
3. aggiungi le utility di input;

Crea il file:

Inserisci il codice seguente:

```java
import java.util.Arrays;
import java.util.Scanner;

public class ToolkitProceduraleIntegrato {

    public static void stampaMenuPrincipale() {
        System.out.println();
        System.out.println("===== TOOLKIT PROCEDURALE INTEGRATO =====");
        System.out.println("1. Area numerica");
        System.out.println("2. Area stringhe");
        System.out.println("0. Esci");
    }

    public static void stampaMenuNumerico() {
        System.out.println();
        System.out.println("----- AREA NUMERICA -----");
        System.out.println("1. Inserisci o sostituisci array");
        System.out.println("2. Stampa array");
        System.out.println("3. Somma e media");
        System.out.println("4. Minimo e massimo");
        System.out.println("5. Cerca valore");
        System.out.println("6. Conta occorrenze");
        System.out.println("7. Stampa copia ordinata crescente");
        System.out.println("8. Stampa copia ordinata decrescente");
        System.out.println("0. Torna al menu principale");
    }

    public static void stampaMenuStringhe() {
        System.out.println();
        System.out.println("----- AREA STRINGHE -----");
        System.out.println("1. Inserisci o sostituisci stringa");
        System.out.println("2. Stampa stringa");
        System.out.println("3. Lunghezza");
        System.out.println("4. Maiuscolo e minuscolo");
        System.out.println("5. Conta vocali");
        System.out.println("6. Conta consonanti");
        System.out.println("7. Conta spazi");
        System.out.println("8. Verifica presenza parola");
        System.out.println("0. Torna al menu principale");
    }

    public static int leggiIntero(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine();

            try {
                return Integer.parseInt(testo);
            } catch (NumberFormatException e) {
                System.out.println("Errore: devi inserire un numero intero valido.");
            }
        }
    }

    public static int leggiInteroPositivo(Scanner scanner, String messaggio) {
        while (true) {
            int valore = leggiIntero(scanner, messaggio);

            if (valore > 0) {
                return valore;
            }

            System.out.println("Errore: il valore deve essere maggiore di zero.");
        }
    }

    public static String leggiStringaNonVuota(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine().trim();

            if (!testo.isEmpty()) {
                return testo;
            }

            System.out.println("Errore: il testo non può essere vuoto.");
        }
    }

    public static int[] leggiArrayInteri(Scanner scanner) {
        int dimensione = leggiInteroPositivo(scanner, "Quanti numeri vuoi inserire? ");
        int[] numeri = new int[dimensione];

        for (int i = 0; i < numeri.length; i++) {
            numeri[i] = leggiIntero(scanner, "Valore in posizione " + i + ": ");
        }

        return numeri;
    }

    public static boolean arrayDisponibile(int[] numeri) {
        return numeri != null;
    }

    public static boolean stringaDisponibile(String testo) {
        return testo != null && !testo.isEmpty();
    }

    public static void stampaArray(int[] numeri) {
        if (!arrayDisponibile(numeri)) {
            System.out.println("Nessun array disponibile. Prima inserisci i dati.");
            return;
        }

        System.out.println("Array: " + Arrays.toString(numeri));
    }

    public static int calcolaSomma(int[] numeri) {
        int somma = 0;

        for (int i = 0; i < numeri.length; i++) {
            somma += numeri[i];
        }

        return somma;
    }

    public static double calcolaMedia(int[] numeri) {
        return (double) calcolaSomma(numeri) / numeri.length;
    }

    public static void stampaSommaMedia(int[] numeri) {
        if (!arrayDisponibile(numeri)) {
            System.out.println("Nessun array disponibile. Prima inserisci i dati.");
            return;
        }

        System.out.println("Somma: " + calcolaSomma(numeri));
        System.out.println("Media: " + calcolaMedia(numeri));
    }

    public static int trovaMinimo(int[] numeri) {
        int minimo = numeri[0];

        for (int i = 1; i < numeri.length; i++) {
            if (numeri[i] < minimo) {
                minimo = numeri[i];
            }
        }

        return minimo;
    }

    public static int trovaMassimo(int[] numeri) {
        int massimo = numeri[0];

        for (int i = 1; i < numeri.length; i++) {
            if (numeri[i] > massimo) {
                massimo = numeri[i];
            }
        }

        return massimo;
    }

    public static void stampaMinimoMassimo(int[] numeri) {
        if (!arrayDisponibile(numeri)) {
            System.out.println("Nessun array disponibile. Prima inserisci i dati.");
            return;
        }

        System.out.println("Minimo: " + trovaMinimo(numeri));
        System.out.println("Massimo: " + trovaMassimo(numeri));
    }

    public static int cercaValore(int[] numeri, int valore) {
        for (int i = 0; i < numeri.length; i++) {
            if (numeri[i] == valore) {
                return i;
            }
        }

        return -1;
    }

    public static void eseguiRicerca(Scanner scanner, int[] numeri) {
        if (!arrayDisponibile(numeri)) {
            System.out.println("Nessun array disponibile. Prima inserisci i dati.");
            return;
        }

        int valore = leggiIntero(scanner, "Valore da cercare: ");
        int posizione = cercaValore(numeri, valore);

        if (posizione >= 0) {
            System.out.println("Valore trovato in posizione: " + posizione);
        } else {
            System.out.println("Valore non trovato.");
        }
    }

    public static int contaOccorrenze(int[] numeri, int valore) {
        int contatore = 0;

        for (int i = 0; i < numeri.length; i++) {
            if (numeri[i] == valore) {
                contatore++;
            }
        }

        return contatore;
    }

    public static void eseguiConteggio(Scanner scanner, int[] numeri) {
        if (!arrayDisponibile(numeri)) {
            System.out.println("Nessun array disponibile. Prima inserisci i dati.");
            return;
        }

        int valore = leggiIntero(scanner, "Valore da contare: ");
        System.out.println("Occorrenze di " + valore + ": " + contaOccorrenze(numeri, valore));
    }

    public static void ordinaCrescente(int[] numeri) {
        for (int i = 0; i < numeri.length - 1; i++) {
            for (int j = i + 1; j < numeri.length; j++) {
                if (numeri[i] > numeri[j]) {
                    int temp = numeri[i];
                    numeri[i] = numeri[j];
                    numeri[j] = temp;
                }
            }
        }
    }

    public static void ordinaDecrescente(int[] numeri) {
        for (int i = 0; i < numeri.length - 1; i++) {
            for (int j = i + 1; j < numeri.length; j++) {
                if (numeri[i] < numeri[j]) {
                    int temp = numeri[i];
                    numeri[i] = numeri[j];
                    numeri[j] = temp;
                }
            }
        }
    }

    public static void stampaCopiaOrdinataCrescente(int[] numeri) {
        if (!arrayDisponibile(numeri)) {
            System.out.println("Nessun array disponibile. Prima inserisci i dati.");
            return;
        }

        int[] copia = Arrays.copyOf(numeri, numeri.length);
        ordinaCrescente(copia);

        System.out.println("Originale: " + Arrays.toString(numeri));
        System.out.println("Crescente: " + Arrays.toString(copia));
    }

    public static void stampaCopiaOrdinataDecrescente(int[] numeri) {
        if (!arrayDisponibile(numeri)) {
            System.out.println("Nessun array disponibile. Prima inserisci i dati.");
            return;
        }

        int[] copia = Arrays.copyOf(numeri, numeri.length);
        ordinaDecrescente(copia);

        System.out.println("Originale: " + Arrays.toString(numeri));
        System.out.println("Decrescente: " + Arrays.toString(copia));
    }

    public static void stampaStringa(String testo) {
        if (!stringaDisponibile(testo)) {
            System.out.println("Nessuna stringa disponibile. Prima inserisci il testo.");
            return;
        }

        System.out.println("Testo: " + testo);
    }

    public static void stampaLunghezza(String testo) {
        if (!stringaDisponibile(testo)) {
            System.out.println("Nessuna stringa disponibile. Prima inserisci il testo.");
            return;
        }

        System.out.println("Lunghezza: " + testo.length());
    }

    public static void stampaMaiuscoloMinuscolo(String testo) {
        if (!stringaDisponibile(testo)) {
            System.out.println("Nessuna stringa disponibile. Prima inserisci il testo.");
            return;
        }

        System.out.println("Maiuscolo: " + testo.toUpperCase());
        System.out.println("Minuscolo: " + testo.toLowerCase());
    }

    public static boolean isVocale(char c) {
        char carattere = Character.toLowerCase(c);
        return carattere == 'a' || carattere == 'e' || carattere == 'i' || carattere == 'o' || carattere == 'u';
    }

    public static int contaVocali(String testo) {
        int contatore = 0;

        for (int i = 0; i < testo.length(); i++) {
            if (isVocale(testo.charAt(i))) {
                contatore++;
            }
        }

        return contatore;
    }

    public static int contaConsonanti(String testo) {
        int contatore = 0;

        for (int i = 0; i < testo.length(); i++) {
            char c = Character.toLowerCase(testo.charAt(i));

            if (c >= 'a' && c <= 'z' && !isVocale(c)) {
                contatore++;
            }
        }

        return contatore;
    }

    public static int contaSpazi(String testo) {
        int contatore = 0;

        for (int i = 0; i < testo.length(); i++) {
            if (testo.charAt(i) == ' ') {
                contatore++;
            }
        }

        return contatore;
    }

    public static void stampaConteggioVocali(String testo) {
        if (!stringaDisponibile(testo)) {
            System.out.println("Nessuna stringa disponibile. Prima inserisci il testo.");
            return;
        }

        System.out.println("Vocali: " + contaVocali(testo));
    }

    public static void stampaConteggioConsonanti(String testo) {
        if (!stringaDisponibile(testo)) {
            System.out.println("Nessuna stringa disponibile. Prima inserisci il testo.");
            return;
        }

        System.out.println("Consonanti: " + contaConsonanti(testo));
    }

    public static void stampaConteggioSpazi(String testo) {
        if (!stringaDisponibile(testo)) {
            System.out.println("Nessuna stringa disponibile. Prima inserisci il testo.");
            return;
        }

        System.out.println("Spazi: " + contaSpazi(testo));
    }

    public static void verificaParola(Scanner scanner, String testo) {
        if (!stringaDisponibile(testo)) {
            System.out.println("Nessuna stringa disponibile. Prima inserisci il testo.");
            return;
        }

        String parola = leggiStringaNonVuota(scanner, "Parola da cercare: ");

        if (testo.toLowerCase().contains(parola.toLowerCase())) {
            System.out.println("La parola è presente nel testo.");
        } else {
            System.out.println("La parola non è presente nel testo.");
        }
    }

    public static int[] gestisciAreaNumerica(Scanner scanner, int[] numeri) {
        int scelta;

        do {
            stampaMenuNumerico();
            scelta = leggiIntero(scanner, "Scelta: ");

            switch (scelta) {
                case 1:
                    numeri = leggiArrayInteri(scanner);
                    System.out.println("Array acquisito correttamente.");
                    break;
                case 2:
                    stampaArray(numeri);
                    break;
                case 3:
                    stampaSommaMedia(numeri);
                    break;
                case 4:
                    stampaMinimoMassimo(numeri);
                    break;
                case 5:
                    eseguiRicerca(scanner, numeri);
                    break;
                case 6:
                    eseguiConteggio(scanner, numeri);
                    break;
                case 7:
                    stampaCopiaOrdinataCrescente(numeri);
                    break;
                case 8:
                    stampaCopiaOrdinataDecrescente(numeri);
                    break;
                case 0:
                    System.out.println("Ritorno al menu principale.");
                    break;
                default:
                    System.out.println("Scelta non valida.");
            }
        } while (scelta != 0);

        return numeri;
    }

    public static String gestisciAreaStringhe(Scanner scanner, String testo) {
        int scelta;

        do {
            stampaMenuStringhe();
            scelta = leggiIntero(scanner, "Scelta: ");

            switch (scelta) {
                case 1:
                    testo = leggiStringaNonVuota(scanner, "Inserisci una stringa: ");
                    System.out.println("Stringa acquisita correttamente.");
                    break;
                case 2:
                    stampaStringa(testo);
                    break;
                case 3:
                    stampaLunghezza(testo);
                    break;
                case 4:
                    stampaMaiuscoloMinuscolo(testo);
                    break;
                case 5:
                    stampaConteggioVocali(testo);
                    break;
                case 6:
                    stampaConteggioConsonanti(testo);
                    break;
                case 7:
                    stampaConteggioSpazi(testo);
                    break;
                case 8:
                    verificaParola(scanner, testo);
                    break;
                case 0:
                    System.out.println("Ritorno al menu principale.");
                    break;
                default:
                    System.out.println("Scelta non valida.");
            }
        } while (scelta != 0);

        return testo;
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int[] numeri = null;
        String testo = null;
        int scelta;

        do {
            stampaMenuPrincipale();
            scelta = leggiIntero(scanner, "Scelta: ");

            switch (scelta) {
                case 1:
                    numeri = gestisciAreaNumerica(scanner, numeri);
                    break;
                case 2:
                    testo = gestisciAreaStringhe(scanner, testo);
                    break;
                case 0:
                    System.out.println("Uscita dal programma.");
                    break;
                default:
                    System.out.println("Scelta non valida.");
            }
        } while (scelta != 0);

        scanner.close();
    }
}
```

## 14. Sintesi

Questa milestone dimostra che puoi costruire un programma più grande anche senza OOP.

Il punto fondamentale è:

```text
main breve, metodi chiari, stato controllato, test espliciti
```

Questo laboratorio prepara il passaggio successivo verso la programmazione a oggetti.
