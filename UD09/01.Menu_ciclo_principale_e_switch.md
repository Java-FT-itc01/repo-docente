# 01. Menu, ciclo principale e `switch`

## Obiettivo

In questa attività imparerai a costruire un programma Java da console guidato da menu.

Un programma con menu:

1. mostra un elenco di operazioni;
2. chiede una scelta all'utente;
3. esegue l'operazione scelta;
4. torna al menu;
5. termina solo quando l'utente sceglie di uscire.

Questo modello sarà usato nel mini-progetto procedurale della UD09.

---

## 1. Prerequisiti

Prima di iniziare dovresti conoscere:

- `Scanner`;
- `nextLine()`;
- `Integer.parseInt(...)`;
- `try/catch`;
- ciclo `while`;
- istruzione `switch`;
- metodi statici.

---

## 2. Menu testuale minimo

Un menu è semplicemente un testo mostrato all'utente.

Esempio:

```text
=== MENU ===
1. Saluta
2. Mostra informazioni
0. Esci
Scelta:
```

Il menu non fa nulla da solo.

Serve un ciclo che lo mostri più volte e una selezione che esegua l'operazione scelta.

---

## 3. Primo programma con menu

Crea un file chiamato:

```text
EsempioMenuBase.java
```

Inserisci:

```java
import java.util.Scanner;

public class EsempioMenuBase {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== MENU ===");
        System.out.println("1. Saluta");
        System.out.println("2. Mostra informazioni");
        System.out.println("0. Esci");
        System.out.print("Scelta: ");

        String scelta = scanner.nextLine();

        System.out.println("Hai scelto: " + scelta);

        scanner.close();
    }
}
```

Compila:

```bash
javac EsempioMenuBase.java
```

Esegui:

```bash
java EsempioMenuBase
```

Questo programma mostra il menu una sola volta.

Non è ancora un vero programma a menu, ma è il primo pezzo.

## 4. Spostare il menu in un metodo

Un programma ordinato evita di lasciare tutto nel `main`.

Creiamo un metodo:

```java
public static void stampaMenu() {
    System.out.println();
    System.out.println("=== MENU ===");
    System.out.println("1. Saluta");
    System.out.println("2. Mostra informazioni");
    System.out.println("0. Esci");
}
```

## 5. Lettura robusta della scelta

La scelta del menu dovrebbe essere un intero.

Useremo una utility già vista:

```java
public static int leggiIntero(Scanner scanner, String messaggio) {
    while (true) {
        System.out.print(messaggio);
        String testo = scanner.nextLine();

        try {
            return Integer.parseInt(testo);
        } catch (NumberFormatException e) {
            System.out.println("Errore: devi inserire un numero intero valido.");
        }
    }
}
```

Questo evita che il programma si blocchi se l'utente scrive:

```text
ciao
```

al posto di un numero.

Per qualche motivo gli utenti lo fanno. E il programma deve sopportarlo.

## 6. Menu con `switch`

Crea un file chiamato:

```text
EsempioMenuSwitch.java
```

Inserisci:

```java
import java.util.Scanner;

public class EsempioMenuSwitch {

    public static void stampaMenu() {
        System.out.println();
        System.out.println("=== MENU ===");
        System.out.println("1. Saluta");
        System.out.println("2. Mostra informazioni");
        System.out.println("0. Esci");
    }

    public static int leggiIntero(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine();

            try {
                return Integer.parseInt(testo);
            } catch (NumberFormatException e) {
                System.out.println("Errore: devi inserire un numero intero valido.");
            }
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        stampaMenu();
        int scelta = leggiIntero(scanner, "Scelta: ");

        switch (scelta) {
            case 1:
                System.out.println("Ciao!");
                break;
            case 2:
                System.out.println("Programma dimostrativo con menu.");
                break;
            case 0:
                System.out.println("Uscita dal programma.");
                break;
            default:
                System.out.println("Scelta non valida.");
        }

        scanner.close();
    }
}
```

Questo programma esegue una scelta, ma poi termina.

Manca il ciclo principale.

---

## 7. Ciclo principale del programma

Un programma con menu deve ripetere il menu finché l'utente non sceglie `0`.

La struttura tipica è:

```java
int scelta;

do {
    stampaMenu();
    scelta = leggiIntero(scanner, "Scelta: ");

    switch (scelta) {
        case 1:
            // operazione 1
            break;
        case 0:
            // uscita
            break;
        default:
            // scelta non valida
    }
} while (scelta != 0);
```

Il ciclo `do-while` è adatto perché il menu deve essere mostrato almeno una volta.

## 8. Programma completo con ciclo menu

Crea un file chiamato:

```text
MenuConCiclo.java
```

Inserisci:

```java
import java.util.Scanner;

public class MenuConCiclo {

    public static void stampaMenu() {
        System.out.println();
        System.out.println("=== MENU ===");
        System.out.println("1. Saluta");
        System.out.println("2. Mostra informazioni");
        System.out.println("0. Esci");
    }

    public static int leggiIntero(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine();

            try {
                return Integer.parseInt(testo);
            } catch (NumberFormatException e) {
                System.out.println("Errore: devi inserire un numero intero valido.");
            }
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int scelta;

        do {
            stampaMenu();
            scelta = leggiIntero(scanner, "Scelta: ");

            switch (scelta) {
                case 1:
                    System.out.println("Ciao!");
                    break;
                case 2:
                    System.out.println("Programma dimostrativo con menu.");
                    break;
                case 0:
                    System.out.println("Uscita dal programma.");
                    break;
                default:
                    System.out.println("Scelta non valida.");
            }
        } while (scelta != 0);

        scanner.close();
    }
}
```

Compila ed esegui.

Prova:

```text
1
2
9
ciao
0
```

Il programma deve:

- eseguire le scelte valide;
- rifiutare l'input non numerico;
- segnalare la scelta fuori menu;
- uscire solo con `0`.

## 9. `do-while` oppure `while`

Il menu può essere scritto anche con `while`.

Esempio:

```java
int scelta = -1;

while (scelta != 0) {
    stampaMenu();
    scelta = leggiIntero(scanner, "Scelta: ");

    switch (scelta) {
        ...
    }
}
```

Entrambe le soluzioni sono accettabili.

In questa UD useremo spesso `do-while` perché il menu deve essere mostrato almeno una volta.

## 10. Errori comuni

### Errore 1: dimenticare di aggiornare la scelta

Se dentro il ciclo non leggi di nuovo la scelta, il programma può diventare infinito.

### Errore 2: dimenticare `break`

Senza `break`, dopo un `case` il programma può continuare nel caso successivo.

### Errore 3: mettere tutta la logica nei `case`

I `case` devono chiamare metodi, non contenere decine di righe.

Meglio:

```java
case 1:
    eseguiOperazione(scanner);
    break;
```

Peggio:

```java
case 1:
    // 80 righe di codice
    break;
```

Il secondo funziona. Poi però serve un archeologo per leggerlo.

## 12. Sintesi

Un programma con menu richiede tre elementi fondamentali:

```text
stampa menu -> leggi scelta -> esegui operazione
```

Questi tre passaggi devono stare dentro un ciclo.

Nel mini-progetto successivo userai questo schema per lavorare con un array e con diverse operazioni.
