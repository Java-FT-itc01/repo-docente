# 02. Stato del programma e array gestito da menu

## Obiettivo

In questa attività capirai un problema tipico dei programmi con menu:

> alcune operazioni richiedono dati già acquisiti.

Esempio:

- opzione 1: inserisci array;
- opzione 2: stampa array;
- opzione 3: calcola somma;
- opzione 4: cerca valore.

Ma cosa succede se l'utente sceglie l'opzione 2 prima dell'opzione 1?

Il programma deve gestire questa situazione.

In caso contrario, rischia di usare un array che non esiste ancora.

---

## 1. Il concetto di stato

Lo stato del programma è l'insieme delle informazioni disponibili in un certo momento.

In un mini-progetto con menu, una variabile può cambiare nel tempo.

Esempio:

```java
int[] numeri = null;
```

All'inizio non abbiamo ancora nessun array.

Dopo l'acquisizione:

```java
numeri = leggiArray(scanner, dimensione);
```

la variabile contiene un riferimento a un array reale.

---

## 2. Che cosa significa `null`

In Java, `null` significa che una variabile riferimento non punta a nessun oggetto o array.

Esempio:

```java
int[] numeri = null;
```

Questa variabile esiste, ma non contiene ancora un array utilizzabile.

Se provi a fare:

```java
System.out.println(numeri.length);
```

mentre `numeri` vale `null`, ottieni una `NullPointerException`.

È Java che dice: "stai usando qualcosa che non c'è". Con il consueto garbo di un allarme antincendio.

---

## 3. Primo esempio problematico

Crea un file chiamato:

```text
ErroreArrayNull.java
```

Inserisci:

```java
public class ErroreArrayNull {
    public static void main(String[] args) {
        int[] numeri = null;

        System.out.println(numeri.length);
    }
}
```

Compila:

```bash
javac ErroreArrayNull.java
```

Esegui:

```bash
java ErroreArrayNull
```

Il programma genera un errore a runtime.

Questo esempio serve a capire perché bisogna controllare lo stato prima di usare l'array.

---

## 4. Metodo `arrayDisponibile`

Per controllare se l'array esiste, possiamo creare un metodo:

```java
public static boolean arrayDisponibile(int[] array) {
    return array != null;
}
```

Se `array` è diverso da `null`, possiamo usarlo.

Se `array` è `null`, dobbiamo mostrare un messaggio e non eseguire l'operazione.

---

## 5. Esempio con controllo

Crea un file chiamato:

```text
ControlloArrayDisponibile.java
```

Inserisci:

```java
public class ControlloArrayDisponibile {

    public static boolean arrayDisponibile(int[] array) {
        return array != null;
    }

    public static void stampaArray(int[] array) {
        if (!arrayDisponibile(array)) {
            System.out.println("Nessun array disponibile. Prima acquisisci i dati.");
            return;
        }

        for (int i = 0; i < array.length; i++) {
            System.out.println(i + ": " + array[i]);
        }
    }

    public static void main(String[] args) {
        int[] numeri = null;

        stampaArray(numeri);

        numeri = new int[] {10, 20, 30};

        stampaArray(numeri);
    }
}
```

Output atteso:

```text
Nessun array disponibile. Prima acquisisci i dati.
0: 10
1: 20
2: 30
```

---

## 6. Il `return` nei metodi `void`

In un metodo `void`, puoi usare:

```java
return;
```

senza valore.

Serve per uscire subito dal metodo.

Esempio:

```java
if (!arrayDisponibile(array)) {
    System.out.println("Nessun array disponibile.");
    return;
}
```

Questo evita di continuare con codice che userebbe un array non valido.

---

## 7. Stato nel menu

In un programma con menu, lo stato viene mantenuto fuori dal ciclo.

Esempio:

```java
int[] numeri = null;

do {
    stampaMenu();
    scelta = leggiIntero(scanner, "Scelta: ");

    switch (scelta) {
        case 1:
            numeri = acquisisciArray(scanner);
            break;
        case 2:
            stampaArray(numeri);
            break;
    }
} while (scelta != 0);
```

La variabile `numeri` viene dichiarata prima del ciclo.

In questo modo mantiene il suo valore tra una scelta e l'altra.

---

## 8. Acquisire o sostituire i dati

Se l'utente sceglie più volte l'opzione di acquisizione, il programma può sostituire l'array precedente.

Esempio:

```java
numeri = acquisisciArray(scanner);
```

Se `numeri` conteneva già un array, il nuovo array prende il posto del precedente.

Per il nostro mini-progetto questo comportamento è accettabile.

---

## 9. Metodo `leggiArray`

Esempio di metodo che crea e restituisce un array:

```java
public static int[] leggiArray(Scanner scanner, int dimensione) {
    int[] array = new int[dimensione];

    for (int i = 0; i < array.length; i++) {
        array[i] = leggiIntero(scanner, "Inserisci valore in posizione " + i + ": ");
    }

    return array;
}
```

Il metodo restituisce l'array completo.

Nel `main` lo useremo così:

```java
numeri = leggiArray(scanner, dimensione);
```

---

## 10. Metodo `acquisisciArray`

Possiamo creare un metodo più completo:

```java
public static int[] acquisisciArray(Scanner scanner) {
    int dimensione = leggiInteroPositivo(scanner, "Quanti numeri vuoi inserire? ");
    return leggiArray(scanner, dimensione);
}
```

Questo metodo:

1. chiede la dimensione;
2. crea e riempie l'array;
3. restituisce l'array.

---

## 11. Esempio completo con menu e stato

Crea un file chiamato:

```text
MenuConArrayStato.java
```

Inserisci:

```java
import java.util.Scanner;
import java.util.Arrays;

public class MenuConArrayStato {

    public static void stampaMenu() {
        System.out.println();
        System.out.println("=== MENU ARRAY ===");
        System.out.println("1. Inserisci array");
        System.out.println("2. Stampa array");
        System.out.println("0. Esci");
    }

    public static int leggiIntero(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine();

            try {
                return Integer.parseInt(testo);
            } catch (NumberFormatException e) {
                System.out.println("Errore: devi inserire un intero valido.");
            }
        }
    }

    public static int leggiInteroPositivo(Scanner scanner, String messaggio) {
        while (true) {
            int valore = leggiIntero(scanner, messaggio);

            if (valore > 0) {
                return valore;
            }

            System.out.println("Errore: il valore deve essere maggiore di zero.");
        }
    }

    public static int[] leggiArray(Scanner scanner, int dimensione) {
        int[] array = new int[dimensione];

        for (int i = 0; i < array.length; i++) {
            array[i] = leggiIntero(scanner, "Inserisci valore in posizione " + i + ": ");
        }

        return array;
    }

    public static int[] acquisisciArray(Scanner scanner) {
        int dimensione = leggiInteroPositivo(scanner, "Quanti numeri vuoi inserire? ");
        return leggiArray(scanner, dimensione);
    }

    public static boolean arrayDisponibile(int[] array) {
        return array != null;
    }

    public static void stampaArray(int[] array) {
        if (!arrayDisponibile(array)) {
            System.out.println("Nessun array disponibile. Prima acquisisci i dati.");
            return;
        }

        System.out.println("Array: " + Arrays.toString(array));
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int[] numeri = null;
        int scelta;

        do {
            stampaMenu();
            scelta = leggiIntero(scanner, "Scelta: ");

            switch (scelta) {
                case 1:
                    numeri = acquisisciArray(scanner);
                    System.out.println("Array acquisito correttamente.");
                    break;
                case 2:
                    stampaArray(numeri);
                    break;
                case 0:
                    System.out.println("Uscita dal programma.");
                    break;
                default:
                    System.out.println("Scelta non valida.");
            }
        } while (scelta != 0);

        scanner.close();
    }
}
```

## 13. Errori comuni

### Errore 1: dichiarare l'array dentro un `case`

Se dichiari l'array dentro `case 1`, gli altri `case` non lo vedono.

Sbagliato:

```java
case 1:
    int[] numeri = acquisisciArray(scanner);
    break;
```

La variabile deve essere dichiarata prima del ciclo.

---

### Errore 2: usare l'array senza controllare `null`

Sbagliato:

```java
System.out.println(numeri.length);
```

se `numeri` può essere ancora `null`.

---

### Errore 3: duplicare il controllo in ogni `case`

Meglio creare metodi come:

```java
stampaArray(numeri)
```

e mettere il controllo dentro il metodo.

## 15. Sintesi

In un programma con menu non basta sapere eseguire le singole operazioni.

Bisogna anche sapere se i dati necessari sono disponibili.

Questo è il punto centrale:

```text
prima acquisisci i dati, poi li usi
```

Nel laboratorio principale userai questa idea per costruire un mini-progetto procedurale completo.
