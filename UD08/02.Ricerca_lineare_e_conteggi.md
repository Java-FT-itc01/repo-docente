# 02. Ricerca lineare e conteggi

## Obiettivo

In questa attività imparerai a cercare valori dentro un array.

L'obiettivo è distinguere bene tre problemi diversi:

1. verificare se un valore è presente;
2. trovare la prima posizione in cui compare;
3. contare quante volte compare.

Sembrano quasi la stessa cosa. Naturalmente non lo sono, perché altrimenti sarebbe troppo comodo.

---

## 1. Prerequisiti

Prima di iniziare dovresti conoscere:

- array;
- indici;
- `array.length`;
- ciclo `for`;
- metodi statici;
- confronto tra interi con `==`;
- input robusto.

---

## 2. Ricerca lineare

La ricerca lineare consiste nel controllare gli elementi dell'array uno alla volta.

Esempio:

```text
Array: [4, 8, 2, 8, 9]
Valore cercato: 8
```

La ricerca procede così:

| Posizione | Valore | Esito |
|---:|---:|---|
| 0 | 4 | non trovato |
| 1 | 8 | trovato |

A questo punto la ricerca può fermarsi.

---

## 3. Metodo `cercaValore`

Crea un file chiamato:

```text
RicercaLineare.java
```

Inserisci:

```java
public class RicercaLineare {

    public static int cercaValore(int[] numeri, int valore) {
        for (int i = 0; i < numeri.length; i++) {
            if (numeri[i] == valore) {
                return i;
            }
        }

        return -1;
    }

    public static void main(String[] args) {
        int[] numeri = {4, 8, 2, 8, 9};

        int posizione = cercaValore(numeri, 8);

        if (posizione >= 0) {
            System.out.println("Valore trovato in posizione: " + posizione);
        } else {
            System.out.println("Valore non trovato.");
        }
    }
}
```

Output atteso:

```text
Valore trovato in posizione: 1
```

---

## 4. Perché restituire `-1`

Un indice valido non può essere negativo.

Quindi `-1` viene spesso usato come valore convenzionale per indicare:

```text
non trovato
```

Il metodo restituisce:

- `0`, `1`, `2`, ... se trova il valore;
- `-1` se non lo trova.

---

## 5. Ricerca con `return`

Nel metodo:

```java
if (numeri[i] == valore) {
    return i;
}
```

quando il valore viene trovato, il metodo termina subito.

Non serve controllare il resto dell'array se l'obiettivo è trovare la prima posizione.

---

## 6. Ricerca con valore assente

Modifica temporaneamente:

```java
int posizione = cercaValore(numeri, 99);
```

Output atteso:

```text
Valore non trovato.
```

Il ciclo controlla tutti gli elementi, non trova il valore e arriva a:

```java
return -1;
```

---

## 7. Conteggio delle occorrenze

Il conteggio è diverso dalla ricerca.

Se vogliamo sapere quante volte compare un valore, dobbiamo visitare tutto l'array.

Esempio:

```text
Array: [4, 8, 2, 8, 9]
Valore: 8
```

Risultato:

```text
2 occorrenze
```

Non possiamo fermarci al primo `8`, perché ce n'è un altro più avanti.

---

## 8. Metodo `contaOccorrenze`

Crea un file chiamato:

```text
ConteggioOccorrenze.java
```

Inserisci:

```java
public class ConteggioOccorrenze {

    public static int contaOccorrenze(int[] numeri, int valore) {
        int contatore = 0;

        for (int i = 0; i < numeri.length; i++) {
            if (numeri[i] == valore) {
                contatore++;
            }
        }

        return contatore;
    }

    public static void main(String[] args) {
        int[] numeri = {4, 8, 2, 8, 9};

        int occorrenze = contaOccorrenze(numeri, 8);

        System.out.println("Occorrenze: " + occorrenze);
    }
}
```

Output atteso:

```text
Occorrenze: 2
```

## 9. Ricerca e conteggio insieme

Spesso vogliamo sapere:

- la prima posizione;
- il numero totale di occorrenze.

Crea un file chiamato:

```text
RicercaEConteggio.java
```

Inserisci:

```java
public class RicercaEConteggio {

    public static int cercaValore(int[] numeri, int valore) {
        for (int i = 0; i < numeri.length; i++) {
            if (numeri[i] == valore) {
                return i;
            }
        }

        return -1;
    }

    public static int contaOccorrenze(int[] numeri, int valore) {
        int contatore = 0;

        for (int i = 0; i < numeri.length; i++) {
            if (numeri[i] == valore) {
                contatore++;
            }
        }

        return contatore;
    }

    public static void stampaRisultato(int valore, int posizione, int occorrenze) {
        if (posizione == -1) {
            System.out.println("Il valore " + valore + " non è presente.");
        } else {
            System.out.println("Il valore " + valore + " compare per la prima volta in posizione " + posizione + ".");
            System.out.println("Occorrenze totali: " + occorrenze);
        }
    }

    public static void main(String[] args) {
        int[] numeri = {4, 8, 2, 8, 9};

        int valore = 8;
        int posizione = cercaValore(numeri, valore);
        int occorrenze = contaOccorrenze(numeri, valore);

        stampaRisultato(valore, posizione, occorrenze);
    }
}
```

## 10. Stampare tutte le posizioni

A volte non basta contare le occorrenze.

Vogliamo anche stampare tutte le posizioni in cui il valore compare.

Esempio:

```java
public static void stampaPosizioni(int[] numeri, int valore) {
    boolean trovato = false;

    for (int i = 0; i < numeri.length; i++) {
        if (numeri[i] == valore) {
            System.out.println("Trovato in posizione: " + i);
            trovato = true;
        }
    }

    if (!trovato) {
        System.out.println("Valore non trovato.");
    }
}
```

La variabile `trovato` serve per sapere se almeno una corrispondenza è stata trovata.

## 11. Versione con input da tastiera

Crea un file chiamato:

```text
RicercaInput.java
```

Inserisci:

```java
import java.util.Scanner;

public class RicercaInput {

    public static int leggiIntero(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine();

            try {
                return Integer.parseInt(testo);
            } catch (NumberFormatException e) {
                System.out.println("Errore: devi inserire un numero intero valido.");
            }
        }
    }

    public static int leggiInteroPositivo(Scanner scanner, String messaggio) {
        while (true) {
            int valore = leggiIntero(scanner, messaggio);

            if (valore > 0) {
                return valore;
            }

            System.out.println("Errore: il valore deve essere maggiore di zero.");
        }
    }

    public static int[] leggiArray(Scanner scanner, int dimensione) {
        int[] numeri = new int[dimensione];

        for (int i = 0; i < numeri.length; i++) {
            numeri[i] = leggiIntero(scanner, "Inserisci valore in posizione " + i + ": ");
        }

        return numeri;
    }

    public static int cercaValore(int[] numeri, int valore) {
        for (int i = 0; i < numeri.length; i++) {
            if (numeri[i] == valore) {
                return i;
            }
        }

        return -1;
    }

    public static int contaOccorrenze(int[] numeri, int valore) {
        int contatore = 0;

        for (int i = 0; i < numeri.length; i++) {
            if (numeri[i] == valore) {
                contatore++;
            }
        }

        return contatore;
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int dimensione = leggiInteroPositivo(scanner, "Quanti numeri vuoi inserire? ");
        int[] numeri = leggiArray(scanner, dimensione);

        int valore = leggiIntero(scanner, "Valore da cercare: ");

        int posizione = cercaValore(numeri, valore);
        int occorrenze = contaOccorrenze(numeri, valore);

        if (posizione >= 0) {
            System.out.println("Prima posizione: " + posizione);
            System.out.println("Occorrenze: " + occorrenze);
        } else {
            System.out.println("Valore non trovato.");
        }

        scanner.close();
    }
}
```

## 13. Errori comuni

### Errore 1: usare `break` nel conteggio

Sbagliato:

```java
if (numeri[i] == valore) {
    contatore++;
    break;
}
```

Il `break` interrompe il ciclo al primo valore trovato.

Nel conteggio non va bene.

---

### Errore 2: dimenticare `return -1`

Se il metodo di ricerca non trova nulla, deve comunque restituire un valore.

```java
return -1;
```

---

### Errore 3: confondere posizione e occorrenze

La posizione indica dove si trova un elemento.

Le occorrenze indicano quante volte compare.

Sono due informazioni diverse.

## 15. Sintesi

Ricerca e conteggio sono due algoritmi diversi.

La ricerca della prima posizione può fermarsi.

Il conteggio deve continuare fino alla fine.

Questa differenza sarà importante quando parleremo di costo degli algoritmi.
