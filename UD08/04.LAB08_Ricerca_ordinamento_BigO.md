# 04. LAB08 - Ricerca, ordinamento e Big O intuitivo

## Obiettivo del laboratorio

In questo laboratorio costruirai un programma che analizza un array di interi.

Il programma dovrà:

1. leggere un array da tastiera;
2. stampare l'array originale;
3. cercare un valore;
4. contare quante volte compare;
5. calcolare minimo, massimo, somma e media;
6. creare una copia dell'array;
7. ordinare la copia in modo crescente;
8. confrontare ricerca e ordinamento dal punto di vista del costo intuitivo.

Questo laboratorio unisce gli argomenti precedenti in un unico programma.

## 4. Metodo `leggiIntero`

Aggiungi questo metodo nella classe:

```java
public static int leggiIntero(Scanner scanner, String messaggio) {
    while (true) {
        System.out.print(messaggio);
        String testo = scanner.nextLine();

        try {
            return Integer.parseInt(testo);
        } catch (NumberFormatException e) {
            System.out.println("Errore: devi inserire un numero intero valido.");
        }
    }
}
```

Questo metodo legge un intero in modo robusto.

## 5. Metodo `leggiInteroPositivo`

Aggiungi:

```java
public static int leggiInteroPositivo(Scanner scanner, String messaggio) {
    while (true) {
        int valore = leggiIntero(scanner, messaggio);

        if (valore > 0) {
            return valore;
        }

        System.out.println("Errore: il valore deve essere maggiore di zero.");
    }
}
```

Questo metodo serve per leggere la dimensione dell'array.

Un array con dimensione zero non è utile per minimo e massimo, perché questi metodi partono da `array[0]`.

E sì, chiedere minimo di niente è un'attività filosofica, non Java.

## 9. Metodo `contaOccorrenze`

Aggiungi:

```java
public static int contaOccorrenze(int[] array, int valore) {
    int contatore = 0;

    for (int i = 0; i < array.length; i++) {
        if (array[i] == valore) {
            contatore++;
        }
    }

    return contatore;
}
```

La differenza rispetto a `cercaValore(...)` è importante:

- `cercaValore(...)` può fermarsi;
- `contaOccorrenze(...)` deve visitare tutto l'array.

## 14. Metodo `ordinaCrescente`

Aggiungi:

```java
public static void ordinaCrescente(int[] array) {
    for (int i = 0; i < array.length - 1; i++) {
        for (int j = i + 1; j < array.length; j++) {
            if (array[i] > array[j]) {
                int temp = array[i];
                array[i] = array[j];
                array[j] = temp;
            }
        }
    }
}
```

Questo metodo modifica l'array ricevuto.

Per non modificare l'array originale, useremo una copia.

## 16. Codice finale completo

Confronta il tuo file con questa versione:

```java
import java.util.Scanner;
import java.util.Arrays;

public class RicercaOrdinamentoBigO {

    public static int leggiIntero(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine();

            try {
                return Integer.parseInt(testo);
            } catch (NumberFormatException e) {
                System.out.println("Errore: devi inserire un numero intero valido.");
            }
        }
    }

    public static int leggiInteroPositivo(Scanner scanner, String messaggio) {
        while (true) {
            int valore = leggiIntero(scanner, messaggio);

            if (valore > 0) {
                return valore;
            }

            System.out.println("Errore: il valore deve essere maggiore di zero.");
        }
    }

    public static int[] leggiArray(Scanner scanner, int dimensione) {
        int[] array = new int[dimensione];

        for (int i = 0; i < array.length; i++) {
            array[i] = leggiIntero(scanner, "Inserisci il valore in posizione " + i + ": ");
        }

        return array;
    }

    public static int cercaValore(int[] array, int valore) {
        for (int i = 0; i < array.length; i++) {
            if (array[i] == valore) {
                return i;
            }
        }

        return -1;
    }

    public static int contaOccorrenze(int[] array, int valore) {
        int contatore = 0;

        for (int i = 0; i < array.length; i++) {
            if (array[i] == valore) {
                contatore++;
            }
        }

        return contatore;
    }

    public static void stampaEsitoRicerca(int valore, int posizione, int occorrenze) {
        if (posizione == -1) {
            System.out.println("Il valore " + valore + " non è presente nell'array.");
        } else {
            System.out.println("Il valore " + valore + " è presente per la prima volta in posizione " + posizione + ".");
            System.out.println("Numero di occorrenze: " + occorrenze);
        }
    }

    public static int trovaMinimo(int[] array) {
        int minimo = array[0];

        for (int i = 1; i < array.length; i++) {
            if (array[i] < minimo) {
                minimo = array[i];
            }
        }

        return minimo;
    }

    public static int trovaMassimo(int[] array) {
        int massimo = array[0];

        for (int i = 1; i < array.length; i++) {
            if (array[i] > massimo) {
                massimo = array[i];
            }
        }

        return massimo;
    }

    public static int calcolaSomma(int[] array) {
        int somma = 0;

        for (int i = 0; i < array.length; i++) {
            somma += array[i];
        }

        return somma;
    }

    public static double calcolaMedia(int[] array) {
        int somma = calcolaSomma(array);
        return (double) somma / array.length;
    }

    public static void ordinaCrescente(int[] array) {
        for (int i = 0; i < array.length - 1; i++) {
            for (int j = i + 1; j < array.length; j++) {
                if (array[i] > array[j]) {
                    int temp = array[i];
                    array[i] = array[j];
                    array[j] = temp;
                }
            }
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("LAB08 - Ricerca, ordinamento e Big O intuitivo");

        int dimensione = leggiInteroPositivo(scanner, "Quanti numeri vuoi inserire? ");
        int[] numeri = leggiArray(scanner, dimensione);

        System.out.println("Array inserito: " + Arrays.toString(numeri));

        int valoreDaCercare = leggiIntero(scanner, "Inserisci il valore da cercare: ");
        int posizione = cercaValore(numeri, valoreDaCercare);
        int occorrenze = contaOccorrenze(numeri, valoreDaCercare);

        stampaEsitoRicerca(valoreDaCercare, posizione, occorrenze);

        int minimo = trovaMinimo(numeri);
        int massimo = trovaMassimo(numeri);
        int somma = calcolaSomma(numeri);
        double media = calcolaMedia(numeri);

        System.out.println("Minimo: " + minimo);
        System.out.println("Massimo: " + massimo);
        System.out.println("Somma: " + somma);
        System.out.println("Media: " + media);

        int[] copiaOrdinata = Arrays.copyOf(numeri, numeri.length);
        ordinaCrescente(copiaOrdinata);

        System.out.println("Array originale: " + Arrays.toString(numeri));
        System.out.println("Array ordinato: " + Arrays.toString(copiaOrdinata));

        scanner.close();
    }
}
```

## 18. Ragionamento Big O intuitivo

Nel file di evidenza spiega:

- ricerca lineare: una scansione, costo circa `n`;
- conteggio: una scansione completa, costo circa `n`;
- minimo/massimo/somma: una scansione, costo circa `n`;
- ordinamento semplice: due cicli annidati, costo circa `n × n`.

Non servono formule avanzate.

Serve capire che ordinare richiede molto più lavoro rispetto a una singola scansione.
