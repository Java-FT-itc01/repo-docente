## 2. Array come sequenza di dati

Un array permette di conservare più valori dello stesso tipo.

Esempio:

```java
int[] numeri = {7, 3, 9, 1, 5};
```

Questo array contiene 5 elementi.

Gli indici validi sono:

```text
0, 1, 2, 3, 4
```

L'ultimo indice è:

```java
numeri.length - 1
```

---

## 3. Scorrere un array

La forma più comune per scorrere un array è:

```java
for (int i = 0; i < numeri.length; i++) {
    System.out.println(numeri[i]);
}
```

La condizione corretta è:

```java
i < numeri.length
```

Non:

```java
i <= numeri.length
```

Se l'array ha 5 elementi, `numeri.length` vale 5, ma l'indice 5 non esiste.

Java qui non dice "quasi giusto". Dice errore. Con una certa freddezza.

## 5. Accumulatore per la somma

Per calcolare la somma degli elementi si usa una variabile accumulatore.

Esempio:

```java
int somma = 0;

for (int i = 0; i < numeri.length; i++) {
    somma += numeri[i];
}
```

La variabile `somma` parte da 0.

A ogni passaggio aggiunge il valore corrente.

```java
public class SommaArray {
```

## 7. Calcolare la media

La media è:

```text
somma / numero di elementi
```

In Java bisogna fare attenzione alla divisione tra interi.

Se scrivi:

```java
int somma = 25;
int n = 2;
System.out.println(somma / n);
```

il risultato è:

```text
12
```

La parte decimale viene persa.

Per ottenere un risultato `double` si usa un cast:

```java
(double) somma / n
```

## 8. Metodo `calcolaMedia`

Aggiungi questo metodo:

```java
public static double calcolaMedia(int[] numeri) {
    int somma = calcolaSomma(numeri);
    return (double) somma / numeri.length;
}
```

Esempio completo:

```java
public class MediaArray {
```

## 9. Trovare il minimo

Per trovare il minimo, non conviene partire da un valore inventato come `0`.

Se l'array contiene solo numeri negativi, partire da `0` può produrre un risultato sbagliato.

Strategia corretta:

1. assumo che il primo elemento sia il minimo;
2. confronto gli altri elementi;
3. aggiorno il minimo solo se trovo un valore più piccolo.

Esempio:

```java
int minimo = numeri[0];

for (int i = 1; i < numeri.length; i++) {
    if (numeri[i] < minimo) {
        minimo = numeri[i];
    }
}
```

Il ciclo parte da `i = 1` perché `numeri[0]` è già stato usato come valore iniziale.

## 11. Lettura robusta di un array

Ora costruiamo un esempio con input da tastiera.

Crea un file chiamato:

```text
StatisticheArrayInput.java
```

Inserisci:

```java
import java.util.Scanner;

public class StatisticheArrayInput {

    public static int leggiIntero(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine();

            try {
                return Integer.parseInt(testo);
            } catch (NumberFormatException e) {
                System.out.println("Errore: devi inserire un numero intero valido.");
            }
        }
    }

    public static int leggiInteroPositivo(Scanner scanner, String messaggio) {
        while (true) {
            int valore = leggiIntero(scanner, messaggio);

            if (valore > 0) {
                return valore;
            }

            System.out.println("Errore: il valore deve essere maggiore di zero.");
        }
    }

    public static int[] leggiArray(Scanner scanner, int dimensione) {
        int[] numeri = new int[dimensione];

        for (int i = 0; i < numeri.length; i++) {
            numeri[i] = leggiIntero(scanner, "Inserisci valore in posizione " + i + ": ");
        }

        return numeri;
    }

    public static int calcolaSomma(int[] numeri) {
        int somma = 0;

        for (int i = 0; i < numeri.length; i++) {
            somma += numeri[i];
        }

        return somma;
    }

    public static double calcolaMedia(int[] numeri) {
        return (double) calcolaSomma(numeri) / numeri.length;
    }

    public static int trovaMinimo(int[] numeri) {
        int minimo = numeri[0];

        for (int i = 1; i < numeri.length; i++) {
            if (numeri[i] < minimo) {
                minimo = numeri[i];
            }
        }

        return minimo;
    }

    public static int trovaMassimo(int[] numeri) {
        int massimo = numeri[0];

        for (int i = 1; i < numeri.length; i++) {
            if (numeri[i] > massimo) {
                massimo = numeri[i];
            }
        }

        return massimo;
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int dimensione = leggiInteroPositivo(scanner, "Quanti numeri vuoi inserire? ");
        int[] numeri = leggiArray(scanner, dimensione);

        System.out.println("Somma: " + calcolaSomma(numeri));
        System.out.println("Media: " + calcolaMedia(numeri));
        System.out.println("Minimo: " + trovaMinimo(numeri));
        System.out.println("Massimo: " + trovaMassimo(numeri));

        scanner.close();
    }
}
```

## 13. Errori comuni

### Errore 1: usare `i <= array.length`

Sbagliato:

```java
for (int i = 0; i <= numeri.length; i++)
```

Corretto:

```java
for (int i = 0; i < numeri.length; i++)
```

---

### Errore 2: inizializzare minimo a `0`

Sbagliato:

```java
int minimo = 0;
```

Meglio:

```java
int minimo = numeri[0];
```

---

### Errore 3: dimenticare il cast nella media

Sbagliato:

```java
return calcolaSomma(numeri) / numeri.length;
```

Corretto:

```java
return (double) calcolaSomma(numeri) / numeri.length;
```
