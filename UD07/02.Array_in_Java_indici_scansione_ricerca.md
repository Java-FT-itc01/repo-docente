# 02. Array in Java: indici, scansione e ricerca

## Obiettivo

In questa attività studierai gli array in Java.

Un array permette di memorizzare più valori dello stesso tipo in una sequenza ordinata.

L'obiettivo è imparare a:

- dichiarare un array;
- creare un array;
- accedere agli elementi tramite indice;
- distinguere tra `String.length()` e `array.length`;
- scorrere un array con `for`;
- scorrere un array con `for-each`;
- cercare un elemento dentro un array.

---

## 1. Che cos'è un array

Un array è una struttura che contiene più elementi dello stesso tipo.

Esempi:

```java
int[] numeri = new int[5];
String[] parole = new String[3];
```

Il primo array può contenere 5 interi.

Il secondo array può contenere 3 stringhe.

Ogni elemento dell'array ha una posizione, chiamata indice.

Gli indici partono da 0.

---

## 2. Indici da zero

In un array di 5 elementi gli indici validi sono:

```text
0, 1, 2, 3, 4
```

Non esiste l'indice `5`.

Questa è una delle prime trappole sugli array. Java ha deciso che contare da zero fosse ragionevole. Noi ci adattiamo, come sempre.

## 4. Errore con indice fuori intervallo

Modifica temporaneamente il programma aggiungendo:

```java
System.out.println(numeri[4]);
```

L'array ha 4 elementi.

Gli indici validi sono:

```text
0, 1, 2, 3
```

L'indice `4` non esiste.

Il programma genera un errore a runtime.

Questo errore è tipico quando si confonde la lunghezza dell'array con l'ultimo indice disponibile.

## 5. Proprietà `length` degli array

Per sapere quanti elementi contiene un array si usa:

```java
array.length
```

Esempio:

```java
public class EsempioArrayLength {
    public static void main(String[] args) {
        int[] numeri = new int[4];

        System.out.println("Lunghezza array: " + numeri.length);
    }
}
```

Output:

```text
Lunghezza array: 4
```

Attenzione:

- per le stringhe si usa `length()`;
- per gli array si usa `length`.

Stringa:

```java
String parola = "java";
System.out.println(parola.length());
```

Array:

```java
int[] numeri = new int[4];
System.out.println(numeri.length);
```

## 6. Scorrere un array con `for`

Il ciclo più usato per scorrere un array è:

```java
for (int i = 0; i < array.length; i++) {
    // usa array[i]
}
```

Esempio:

```java
public class ScansioneArrayFor {
    public static void main(String[] args) {
        int[] numeri = new int[4];

        numeri[0] = 10;
        numeri[1] = 20;
        numeri[2] = 30;
        numeri[3] = 40;

        for (int i = 0; i < numeri.length; i++) {
            System.out.println("Indice " + i + ": " + numeri[i]);
        }
    }
}
```

---

## 7. Perché usare `i < array.length`

La condizione corretta è:

```java
i < array.length
```

Non:

```java
i <= array.length
```

Se `array.length` vale 4, gli indici validi sono:

```text
0, 1, 2, 3
```

Con `i <= array.length`, il ciclo arriverebbe anche a `i = 4`, che non è valido.

---

## 8. Scorrere un array con `for-each`

Quando non ti serve l'indice, puoi usare il ciclo `for-each`.

Esempio:

```java
public class ScansioneArrayForEach {
    public static void main(String[] args) {
        String[] parole = {"java", "array", "string"};

        for (String parola : parole) {
            System.out.println(parola);
        }
    }
}
```

Il `for-each` è più leggibile, ma non ti dà direttamente l'indice.

Se ti serve sapere la posizione, usa il `for` classico.

## 9. Riempire un array da tastiera

Ora usiamo `Scanner`.

Crea un file chiamato:

```text
RiempiArrayParole.java
```

Inserisci il codice:

```java
import java.util.Scanner;

public class RiempiArrayParole {

    public static String leggiStringaNonVuota(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine().trim();

            if (!testo.isEmpty()) {
                return testo;
            }

            System.out.println("Errore: il testo non può essere vuoto.");
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        String[] parole = new String[3];

        for (int i = 0; i < parole.length; i++) {
            parole[i] = leggiStringaNonVuota(scanner, "Inserisci parola " + i + ": ");
        }

        System.out.println("Parole inserite:");

        for (int i = 0; i < parole.length; i++) {
            System.out.println(i + ": " + parole[i]);
        }

        scanner.close();
    }
}
```

## 10. Array di interi con input robusto

Crea un file chiamato:

```text
ArrayInteriInputRobusto.java
```

Inserisci il codice:

```java
import java.util.Scanner;

public class ArrayInteriInputRobusto {

    public static int leggiIntero(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine();

            try {
                return Integer.parseInt(testo);
            } catch (NumberFormatException e) {
                System.out.println("Errore: devi inserire un intero valido.");
            }
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int[] numeri = new int[5];

        for (int i = 0; i < numeri.length; i++) {
            numeri[i] = leggiIntero(scanner, "Inserisci numero in posizione " + i + ": ");
        }

        int somma = 0;

        for (int i = 0; i < numeri.length; i++) {
            somma += numeri[i];
        }

        double media = (double) somma / numeri.length;

        System.out.println("Somma: " + somma);
        System.out.println("Media: " + media);

        scanner.close();
    }
}
```

## 11. Ricerca lineare in un array di interi

La ricerca lineare controlla gli elementi uno alla volta.

Crea un file chiamato:

```text
RicercaNumeroArray.java
```

Inserisci il codice:

```java
public class RicercaNumeroArray {

    public static int cercaNumero(int[] numeri, int cercato) {
        for (int i = 0; i < numeri.length; i++) {
            if (numeri[i] == cercato) {
                return i;
            }
        }

        return -1;
    }

    public static void main(String[] args) {
        int[] numeri = {10, 20, 30, 40};

        int posizione = cercaNumero(numeri, 30);

        if (posizione >= 0) {
            System.out.println("Numero trovato in posizione: " + posizione);
        } else {
            System.out.println("Numero non trovato.");
        }
    }
}
```

Il valore `-1` viene usato per indicare che l'elemento non è stato trovato.

## 12. Ricerca in un array di stringhe

Per cercare una stringa bisogna usare `equals(...)` o `equalsIgnoreCase(...)`.

Crea un file chiamato:

```text
RicercaParolaArray.java
```

Inserisci il codice:

```java
public class RicercaParolaArray {

    public static int cercaParolaIgnoreCase(String[] parole, String cercata) {
        for (int i = 0; i < parole.length; i++) {
            if (parole[i].equalsIgnoreCase(cercata)) {
                return i;
            }
        }

        return -1;
    }

    public static void main(String[] args) {
        String[] parole = {"java", "array", "scanner"};

        int posizione = cercaParolaIgnoreCase(parole, "ARRAY");

        if (posizione >= 0) {
            System.out.println("Parola trovata in posizione: " + posizione);
        } else {
            System.out.println("Parola non trovata.");
        }
    }
}
```

## 13. Errori comuni

### Errore 1

```java
for (int i = 0; i <= array.length; i++)
```

Corretto:

```java
for (int i = 0; i < array.length; i++)
```

---

### Errore 2

```java
array.length()
```

Corretto:

```java
array.length
```

---

### Errore 3

```java
parola.length
```

Corretto:

```java
parola.length()
```

---

### Errore 4

```java
if (parole[i] == cercata)
```

Corretto:

```java
if (parole[i].equals(cercata))
```

oppure:

```java
if (parole[i].equalsIgnoreCase(cercata))
```

## 16. Sintesi

Gli array permettono di memorizzare sequenze di dati dello stesso tipo.

La forma più importante da ricordare è:

```java
for (int i = 0; i < array.length; i++) {
    // usa array[i]
}
```

Questa struttura sarà usata nel laboratorio principale per leggere, stampare e cercare parole in un array.
