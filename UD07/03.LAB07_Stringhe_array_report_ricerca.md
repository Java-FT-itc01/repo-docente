# 03. LAB07 - Stringhe, array, report e ricerca

## Obiettivo del laboratorio

In questo laboratorio costruirai un programma Java che lavora con un array di parole.

Il programma dovrà:

1. chiedere quante parole inserire;
2. creare un array `String[]`;
3. leggere le parole da tastiera;
4. stampare un report per ogni parola;
5. chiedere una parola da cercare;
6. cercarla ignorando maiuscole e minuscole;
7. indicare se la parola è stata trovata e in quale posizione.

Questo laboratorio integra:

- input robusto;
- metodi statici;
- `String`;
- array;
- ciclo `for`;
- confronto tra stringhe;
- ricerca lineare.

## 4. Metodo `leggiInteroInRange`

Per evitare array troppo grandi o quantità non valide, useremo un metodo che legge un intero compreso tra un minimo e un massimo.

Aggiungi questo metodo nella classe:

```java
public static int leggiInteroInRange(Scanner scanner, String messaggio, int minimo, int massimo) {
    while (true) {
        System.out.print(messaggio);
        String testo = scanner.nextLine();

        try {
            int valore = Integer.parseInt(testo);

            if (valore >= minimo && valore <= massimo) {
                return valore;
            }

            System.out.println("Errore: inserisci un valore tra " + minimo + " e " + massimo + ".");
        } catch (NumberFormatException e) {
            System.out.println("Errore: devi inserire un intero valido.");
        }
    }
}
```

Questo metodo:

- legge una riga;
- prova a convertirla in `int`;
- controlla che sia nel range indicato;
- ripete la richiesta finché il valore non è valido.

## 5. Metodo `leggiStringaNonVuota`

Aggiungi questo metodo:

```java
public static String leggiStringaNonVuota(Scanner scanner, String messaggio) {
    while (true) {
        System.out.print(messaggio);
        String testo = scanner.nextLine().trim();

        if (!testo.isEmpty()) {
            return testo;
        }

        System.out.println("Errore: il testo non può essere vuoto.");
    }
}
```

Questo metodo evita di accettare stringhe vuote o composte solo da spazi.

## 11. Metodo `cercaParolaIgnoreCase`

Aggiungi:

```java
public static int cercaParolaIgnoreCase(String[] parole, String cercata) {
    for (int i = 0; i < parole.length; i++) {
        if (parole[i].equalsIgnoreCase(cercata)) {
            return i;
        }
    }

    return -1;
}
```

Il metodo restituisce:

- la posizione della prima parola trovata;
- `-1` se la parola non è presente.

import java.util.Scanner;

public class Lab07ParoleArray {

    public static int leggiInteroInRange(Scanner scanner, String messaggio, int minimo, int massimo) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine();

            try {
                int valore = Integer.parseInt(testo);

                if (valore >= minimo && valore <= massimo) {
                    return valore;
                }

                System.out.println("Errore: inserisci un valore tra " + minimo + " e " + massimo + ".");
            } catch (NumberFormatException e) {
                System.out.println("Errore: devi inserire un intero valido.");
            }
        }
    }

    public static String leggiStringaNonVuota(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine().trim();

            if (!testo.isEmpty()) {
                return testo;
            }

            System.out.println("Errore: il testo non può essere vuoto.");
        }
    }

    public static void riempiArrayParole(Scanner scanner, String[] parole) {
        for (int i = 0; i < parole.length; i++) {
            parole[i] = leggiStringaNonVuota(scanner, "Inserisci parola in posizione " + i + ": ");
        }
    }

    public static void stampaReportParole(String[] parole) {
        System.out.println();
        System.out.println("=== REPORT PAROLE ===");

        for (int i = 0; i < parole.length; i++) {
            System.out.println(
                "Posizione: " + i +
                " | Parola: " + parole[i] +
                " | Lunghezza: " + parole[i].length() +
                " | Maiuscolo: " + parole[i].toUpperCase()
            );
        }
    }

    public static int cercaParolaIgnoreCase(String[] parole, String cercata) {
        for (int i = 0; i < parole.length; i++) {
            if (parole[i].equalsIgnoreCase(cercata)) {
                return i;
            }
        }

        return -1;
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("LAB07 - Stringhe e array");

        int quantita = leggiInteroInRange(scanner, "Quante parole vuoi inserire? ", 1, 20);
        String[] parole = new String[quantita];

        riempiArrayParole(scanner, parole);
        stampaReportParole(parole);

        String parolaDaCercare = leggiStringaNonVuota(scanner, "Inserisci una parola da cercare: ");
        int posizione = cercaParolaIgnoreCase(parole, parolaDaCercare);

        if (posizione >= 0) {
            System.out.println("Parola trovata in posizione: " + posizione);
        } else {
            System.out.println("Parola non trovata.");
        }

        scanner.close();
    }
