# 04. Esercizi LAB07 - Stringhe e array

## Obiettivo degli esercizi

Questi esercizi servono a consolidare l'uso di:

- `String`;
- metodi fondamentali delle stringhe;
- array di interi;
- array di stringhe;
- ciclo `for`;
- ciclo `for-each`;
- ricerca lineare;
- conteggi;
- report semplici;
- metodi statici.

Gli esercizi sono divisi in tre livelli:

- base;
- intermedi;
- challenge.

Non è necessario svolgerli tutti in aula.

Il file può essere usato come banca esercizi.

---

## 1. Regole generali

Per tutti gli esercizi:

1. crea un solo file `.java` per esercizio;
2. usa una sola classe per file;
3. usa metodi statici quando una logica è riusabile;
4. leggi le stringhe con `nextLine()`;
5. usa `trim()` quando vuoi evitare stringhe vuote;
6. usa `equals(...)` o `equalsIgnoreCase(...)` per confrontare stringhe;
7. usa `array.length` per gli array;
8. usa `String.length()` per le stringhe;
9. usa `i < array.length` nei cicli con indice;
10. non usare `==` per confrontare stringhe.

# Esercizi base

## Esercizio 1 - Report di una parola

### File da creare

```text
Esercizio01_ReportParola.java
```

### Richiesta

Scrivi un programma che:

- legga una parola non vuota;
- stampi la parola originale;
- stampi la lunghezza;
- stampi il primo carattere;
- stampi il secondo carattere solo se esiste;
- stampi la parola in maiuscolo;
- stampi la parola in minuscolo.

### Metodi richiesti

```java
public static String leggiStringaNonVuota(Scanner scanner, String messaggio)
```

```java
public static void stampaReport(String parola)
```

### Test minimi

| Input | Verifica |
|---|---|
| `Java` | lunghezza 4, primo carattere J |
| `A` | non deve generare errore su `charAt(1)` |
| stringa vuota | deve essere rifiutata |

## Esercizio 4 - Ricerca di un numero

### File da creare

```text
Esercizio04_RicercaNumero.java
```

### Richiesta

Scrivi un programma che:

- legga 6 interi;
- li memorizzi in un array;
- chieda un numero da cercare;
- stampi la posizione della prima occorrenza;
- stampi un messaggio se il numero non è presente.

### Metodo richiesto

```java
public static int cercaNumero(int[] numeri, int cercato)
```

Il metodo deve restituire:

- posizione trovata;
- `-1` se non trova il numero.

### Test minimi

Array:

```text
10 20 30 40 50 60
```

Cerca:

```text
30
```

Risultato atteso:

```text
Numero trovato in posizione: 2
```

Cerca:

```text
99
```

Risultato atteso:

```text
Numero non trovato.
```

# Sintesi

Questi esercizi consolidano il passaggio da programmi che lavorano su singoli valori a programmi che lavorano su sequenze.

Con gli array puoi:

- memorizzare più valori;
- scorrerli;
- cercare elementi;
- calcolare report;
- costruire piccoli algoritmi.

Questo è un passaggio fondamentale prima di affrontare collezioni, ordinamenti e strutture dati più avanzate.
