# 01. Stringhe in Java: metodi fondamentali e confronti

## Obiettivo

In questa attività studierai la classe `String` e alcuni metodi fondamentali per lavorare con sequenze di caratteri.

Prima di usare array di parole, ricerche e piccoli report, è necessario capire bene come funziona una stringa.

L'obiettivo è saper usare:

- `length()`;
- `charAt(...)`;
- `trim()`;
- `toUpperCase()`;
- `toLowerCase()`;
- `equals(...)`;
- `equalsIgnoreCase(...)`.

Imparerai anche perché confrontare stringhe con `==` è un errore frequente.

---

## 1. Che cos'è una stringa

In Java una stringa è un oggetto della classe `String`.

Una stringa rappresenta una sequenza di caratteri.

Esempi:

```java
String nome = "Luca";
String citta = "Napoli";
String vuota = "";
```

Una stringa può contenere:

- lettere;
- numeri;
- spazi;
- simboli;
- nessun carattere.

Esempio:

```java
String testo = "Java 21!";
System.out.println(testo);
```

## 3. Metodo `length()`

Il metodo:

```java
length()
```

restituisce il numero di caratteri presenti nella stringa.

Esempio:

```java
public class EsempioLength {
    public static void main(String[] args) {
        String parola = "casa";

        System.out.println(parola.length());
    }
}
```

Output:

```text
4
```

La parola `"casa"` contiene quattro caratteri.

Attenzione: `length()` è un metodo, quindi usa le parentesi.

Corretto:

```java
parola.length()
```

Sbagliato:

```java
parola.length
```

Per una stringa servono le parentesi. Java, nel dubbio, non perdona.

---

## 4. Metodo `charAt(...)`

Il metodo:

```java
charAt(indice)
```

restituisce il carattere che si trova in una certa posizione.

Gli indici partono da zero.

Esempio:

```java
public class EsempioCharAt {
    public static void main(String[] args) {
        String parola = "casa";

        System.out.println(parola.charAt(0));
        System.out.println(parola.charAt(1));
        System.out.println(parola.charAt(2));
        System.out.println(parola.charAt(3));
    }
}
```

Output:

```text
c
a
s
a
```

Per la stringa:

```text
casa
```

gli indici sono:

| Indice | Carattere |
|---:|---|
| 0 | c |
| 1 | a |
| 2 | s |
| 3 | a |

---

## 5. Errore con indice fuori intervallo

Se provi a leggere un indice non valido, il programma va in errore.

Esempio:

```java
public class ErroreCharAt {
    public static void main(String[] args) {
        String parola = "casa";

        System.out.println(parola.charAt(4));
    }
}
```

La stringa `"casa"` ha lunghezza 4, ma gli indici validi sono:

```text
0, 1, 2, 3
```

L'indice `4` non esiste.

Questo genera un errore a runtime.

---

## 6. Controllare la lunghezza prima di usare `charAt`

Prima di usare `charAt(1)`, devi essere sicuro che la stringa abbia almeno due caratteri.

Esempio:

```java
public class ControlloCharAt {
    public static void main(String[] args) {
        String parola = "A";

        if (parola.length() >= 2) {
            System.out.println("Secondo carattere: " + parola.charAt(1));
        } else {
            System.out.println("La parola non ha un secondo carattere.");
        }
    }
}
```

Questo controllo evita l'accesso a una posizione inesistente.

## 7. Metodi `toUpperCase()` e `toLowerCase()`

Questi metodi restituiscono una nuova stringa:

```java
toUpperCase()
toLowerCase()
```

Esempio:

```java
public class EsempioMaiuscoleMinuscole {
    public static void main(String[] args) {
        String parola = "Java";

        System.out.println(parola.toUpperCase());
        System.out.println(parola.toLowerCase());
        System.out.println(parola);
    }
}
```

Output:

```text
JAVA
java
Java
```

La stringa originale non viene modificata.

I metodi restituiscono nuove stringhe.

## 8. Metodo `trim()`

Il metodo:

```java
trim()
```

elimina gli spazi iniziali e finali.

Esempio:

```java
public class EsempioTrim {
    public static void main(String[] args) {
        String testo = "   Java   ";

        System.out.println("[" + testo + "]");
        System.out.println("[" + testo.trim() + "]");
    }
}
```

Output:

```text
[   Java   ]
[Java]
```

`trim()` è molto utile quando si leggono dati da tastiera.

Un utente può inserire spazi prima o dopo una parola.

Naturalmente non lo farà mai apposta, ma il software deve essere pronto anche alla creatività del dito umano.

## 9. Leggere una stringa non vuota

Ora creiamo un metodo statico utile.

Crea un file chiamato:

```text
LeggiStringaNonVuota.java
```

Inserisci il codice:

```java
import java.util.Scanner;

public class LeggiStringaNonVuota {

    public static String leggiStringaNonVuota(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine().trim();

            if (!testo.isEmpty()) {
                return testo;
            }

            System.out.println("Errore: il testo non può essere vuoto.");
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        String nome = leggiStringaNonVuota(scanner, "Inserisci il tuo nome: ");

        System.out.println("Nome acquisito: " + nome);

        scanner.close();
    }
}
```

Questo metodo:

- legge una riga;
- elimina spazi iniziali e finali;
- controlla che il risultato non sia vuoto;
- ripete la richiesta se il testo non è valido.

## 10. Confrontare stringhe con `equals(...)`

Per confrontare il contenuto di due stringhe si usa:

```java
equals(...)
```

Esempio:

```java
public class EsempioEquals {
    public static void main(String[] args) {
        String parola = "java";

        if (parola.equals("java")) {
            System.out.println("La parola coincide.");
        } else {
            System.out.println("La parola non coincide.");
        }
    }
}
```

Output:

```text
La parola coincide.
```

---

## 11. Confrontare ignorando maiuscole e minuscole

Per ignorare la differenza tra maiuscole e minuscole si usa:

```java
equalsIgnoreCase(...)
```

Esempio:

```java
public class EsempioEqualsIgnoreCase {
    public static void main(String[] args) {
        String parola = "Java";

        if (parola.equalsIgnoreCase("java")) {
            System.out.println("La parola coincide ignorando maiuscole e minuscole.");
        } else {
            System.out.println("La parola non coincide.");
        }
    }
}
```

Output:

```text
La parola coincide ignorando maiuscole e minuscole.
```

---

## 12. Perché non usare `==` con le stringhe

Con le stringhe, `==` non è il modo corretto per confrontare il contenuto.

Usa:

```java
equals(...)
```

oppure:

```java
equalsIgnoreCase(...)
```

Esempio consigliato:

```java
if (parola.equals("java")) {
    System.out.println("Uguale");
}
```

Esempio da evitare:

```java
if (parola == "java") {
    System.out.println("Uguale");
}
```

Il motivo completo riguarda oggetti e riferimenti.

Per ora tieni questa regola:

> per confrontare il contenuto di due stringhe, usa `equals(...)`.

## 13. Esercizio guidato: report di una parola

Crea un file chiamato:

```text
ReportParola.java
```

Il programma deve:

1. leggere una parola non vuota;
2. stampare la parola originale;
3. stampare la lunghezza;
4. stampare il primo carattere;
5. stampare il secondo carattere solo se esiste;
6. stampare la parola in maiuscolo;
7. stampare la parola in minuscolo.

Codice completo:

```java
import java.util.Scanner;

public class ReportParola {

    public static String leggiStringaNonVuota(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine().trim();

            if (!testo.isEmpty()) {
                return testo;
            }

            System.out.println("Errore: il testo non può essere vuoto.");
        }
    }

    public static void stampaReport(String parola) {
        System.out.println("Parola: " + parola);
        System.out.println("Lunghezza: " + parola.length());
        System.out.println("Primo carattere: " + parola.charAt(0));

        if (parola.length() >= 2) {
            System.out.println("Secondo carattere: " + parola.charAt(1));
        } else {
            System.out.println("Secondo carattere non disponibile.");
        }

        System.out.println("Maiuscolo: " + parola.toUpperCase());
        System.out.println("Minuscolo: " + parola.toLowerCase());
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        String parola = leggiStringaNonVuota(scanner, "Inserisci una parola: ");
        stampaReport(parola);

        scanner.close();
    }
}
```
