## Note didattiche di conversione

- Le immagini non necessarie ai contenuti didattici non sono state mantenute.
- Gli schemi concettuali sono stati riscritti come diagrammi renderizzabili.
- Alcuni termini sono stati normalizzati:
  - `Structured Query Language` al posto di `Structure Query Language`.
  - `Transaction Log` al posto di `Transection Log`.
- I contenuti sono stati resi più lineari per l'uso in aula o in repository didattico.
