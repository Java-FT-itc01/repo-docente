## 2. File `Libro.java`

Crea:

```text
src/corso/lab12/Libro.java
```

Inserisci:

```java
package corso.lab12;

public class Libro {
    private String titolo;
    private String autore;
    private double prezzo;
    private int numeroPagine;

    private static int numeroLibriCreati = 0;

    public static final double IVA_LIBRI = 0.04;
    public static final int SOGLIA_LIBRO_LUNGO = 300;

    public Libro(String titolo, String autore, double prezzo, int numeroPagine) {
        this.titolo = "Titolo non disponibile";
        this.autore = "Autore non disponibile";
        this.prezzo = 0.0;
        this.numeroPagine = 1;

        setTitolo(titolo);
        setAutore(autore);
        setPrezzo(prezzo);
        setNumeroPagine(numeroPagine);

        numeroLibriCreati++;
    }

    public String getTitolo() {
        return titolo;
    }

    public void setTitolo(String titolo) {
        if (titolo != null && !titolo.trim().isEmpty()) {
            this.titolo = titolo.trim();
        } else {
            System.out.println("Titolo non valido: il valore precedente resta invariato.");
        }
    }

    public String getAutore() {
        return autore;
    }

    public void setAutore(String autore) {
        if (autore != null && !autore.trim().isEmpty()) {
            this.autore = autore.trim();
        } else {
            System.out.println("Autore non valido: il valore precedente resta invariato.");
        }
    }

    public double getPrezzo() {
        return prezzo;
    }

    public void setPrezzo(double prezzo) {
        if (prezzo >= 0) {
            this.prezzo = prezzo;
        } else {
            System.out.println("Prezzo non valido: il valore precedente resta invariato.");
        }
    }

    public int getNumeroPagine() {
        return numeroPagine;
    }

    public void setNumeroPagine(int numeroPagine) {
        if (numeroPagine > 0) {
            this.numeroPagine = numeroPagine;
        } else {
            System.out.println("Numero pagine non valido: il valore precedente resta invariato.");
        }
    }

    public double calcolaPrezzoConIva() {
        return prezzo + prezzo * IVA_LIBRI;
    }

    public double calcolaPrezzoScontato(double percentualeSconto) {
        if (percentualeSconto < 0 || percentualeSconto > 100) {
            return prezzo;
        }

        return prezzo - prezzo * percentualeSconto / 100;
    }

    public boolean libroLungo() {
        return numeroPagine >= SOGLIA_LIBRO_LUNGO;
    }

    public void stampaScheda() {
        System.out.println("Titolo: " + titolo);
        System.out.println("Autore: " + autore);
        System.out.println("Prezzo: " + prezzo);
        System.out.println("Prezzo con IVA: " + calcolaPrezzoConIva());
        System.out.println("Numero pagine: " + numeroPagine);
        System.out.println("Libro lungo: " + libroLungo());
    }

    public static int getNumeroLibriCreati() {
        return numeroLibriCreati;
    }
}
```

## 3. Analisi della classe `Libro`

La classe usa:

```java
private String titolo;
private String autore;
private double prezzo;
private int numeroPagine;
```

Gli attributi sono privati.

Questo significa che non possono essere modificati direttamente dall'esterno.

Non puoi scrivere da `AppLibro`:

```java
libro.prezzo = -100;
```

Devi usare:

```java
libro.setPrezzo(...)
```

Così la classe può controllare se il valore è valido.

---

## 4. Costanti `static final`

La classe contiene:

```java
public static final double IVA_LIBRI = 0.04;
public static final int SOGLIA_LIBRO_LUNGO = 300;
```

Questi valori:

- appartengono alla classe;
- non cambiano;
- hanno un nome significativo.

Sono usati nei metodi:

```java
calcolaPrezzoConIva()
libroLungo()
```

---

## 5. Contatore `static`

La classe contiene:

```java
private static int numeroLibriCreati = 0;
```

Nel costruttore:

```java
numeroLibriCreati++;
```

Ogni volta che viene creato un libro, il contatore aumenta.

Il valore viene letto con:

```java
public static int getNumeroLibriCreati()
```

e usato così:

```java
Libro.getNumeroLibriCreati()
```

## 6. File `AppLibro.java`

Crea:

```text
src/corso/lab12/AppLibro.java
```

Inserisci:

```java
package corso.lab12;

import java.util.Scanner;

public class AppLibro {

    public static String leggiStringaNonVuota(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine().trim();

            if (!testo.isEmpty()) {
                return testo;
            }

            System.out.println("Errore: il testo non può essere vuoto.");
        }
    }

    public static double leggiDouble(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine();

            try {
                Double valore = Double.valueOf(testo);
                return valore.doubleValue();
            } catch (NumberFormatException e) {
                System.out.println("Errore: devi inserire un numero reale valido.");
            }
        }
    }

    public static int leggiIntero(Scanner scanner, String messaggio) {
        while (true) {
            System.out.print(messaggio);
            String testo = scanner.nextLine();

            try {
                Integer valore = Integer.valueOf(testo);
                return valore.intValue();
            } catch (NumberFormatException e) {
                System.out.println("Errore: devi inserire un numero intero valido.");
            }
        }
    }

    public static Libro creaLibroDaInput(Scanner scanner) {
        String titolo = leggiStringaNonVuota(scanner, "Titolo: ");
        String autore = leggiStringaNonVuota(scanner, "Autore: ");
        double prezzo = leggiDouble(scanner, "Prezzo: ");
        int numeroPagine = leggiIntero(scanner, "Numero pagine: ");

        return new Libro(titolo, autore, prezzo, numeroPagine);
    }

    public static void provaSetter(Libro libro) {
        System.out.println();
        System.out.println("=== TEST SETTER ===");

        System.out.println("Prezzo iniziale: " + libro.getPrezzo());

        libro.setPrezzo(-50);
        System.out.println("Dopo setPrezzo(-50): " + libro.getPrezzo());

        libro.setPrezzo(35.90);
        System.out.println("Dopo setPrezzo(35.90): " + libro.getPrezzo());

        libro.setNumeroPagine(0);
        System.out.println("Dopo setNumeroPagine(0): " + libro.getNumeroPagine());
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        Libro libro1 = new Libro("Fondamenti di Java", "Mario Rossi", 39.90, 280);
        Libro libro2 = new Libro("OOP con Java", "Anna Verdi", 49.50, 420);

        System.out.println("=== LIBRO 1 ===");
        libro1.stampaScheda();

        System.out.println();
        System.out.println("=== LIBRO 2 ===");
        libro2.stampaScheda();

        provaSetter(libro1);

        System.out.println();
        System.out.println("=== CREA UN LIBRO DA INPUT ===");
        Libro libro3 = creaLibroDaInput(scanner);

        System.out.println();
        System.out.println("=== LIBRO 3 ===");
        libro3.stampaScheda();

        System.out.println();
        System.out.println("Libri creati: " + Libro.getNumeroLibriCreati());
        System.out.println("IVA libri: " + Libro.IVA_LIBRI);
        System.out.println("Soglia libro lungo: " + Libro.SOGLIA_LIBRO_LUNGO);

        scanner.close();
    }
}
```

## 7. Wrapper nel parsing

Nel metodo `leggiDouble(...)` viene usato:

```java
Double valore = Double.valueOf(testo);
return valore.doubleValue();
```

Nel metodo `leggiIntero(...)` viene usato:

```java
Integer valore = Integer.valueOf(testo);
return valore.intValue();
```

Questo collega il parsing già visto con le wrapper class.

In molti programmi userai anche:

```java
Double.parseDouble(...)
Integer.parseInt(...)
```

Qui usiamo `valueOf(...)` per mostrare esplicitamente il wrapper.

## 13. Errori comuni

### Errore 1 - Dimenticare `private`

Sbagliato:

```java
public double prezzo;
```

Corretto:

```java
private double prezzo;
```

---

### Errore 2 - Setter senza controllo

Debole:

```java
public void setPrezzo(double prezzo) {
    this.prezzo = prezzo;
}
```

Meglio:

```java
public void setPrezzo(double prezzo) {
    if (prezzo >= 0) {
        this.prezzo = prezzo;
    }
}
```

---

### Errore 3 - Rendere statico un attributo dell'oggetto

Sbagliato:

```java
private static String titolo;
```

Corretto:

```java
private String titolo;
```

---

Corretto:

